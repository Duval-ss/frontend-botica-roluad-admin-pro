# Backend Botica ROLUAD

API REST para e-commerce de botica usando Node.js, Express, MySQL, dotenv, cors, bcrypt y jsonwebtoken.

## Pasos

1. Crear la base de datos ejecutando en MySQL/phpMyAdmin:

```sql
sql/botica_roluad.sql
```

2. Instalar dependencias:

```bash
npm install
```

3. Configurar `.env` si tu MySQL tiene contraseña.

4. Ejecutar:

```bash
npm run dev
```
